# React Native APP

A Test app made with React-Native 0.69

## Main technologies used

1. Redux Toolkit + Saga
2. Splash Screen
3. StoryBook
4. Styled Components
5. Jest + Enzyme
6. React Navigation

## Install and run

- Make sure to have install JDK 11 for Android

### Android steps

- yarn install
- npx react-native run-android

### Ios steps

- yarn install
- npx pod-install
- npx react-native run-ios
